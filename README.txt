== Description ==
Connect Turpial App platform with your WooCommerce store for seamless order management and processing. Specialized in AFIP (Argentina) and SENIAT (Venezuela) tax compliance and electronic invoicing.

Turpial App for WooCommerce is a powerful integration plugin that connects your WooCommerce store with the Turpial App platform. This plugin allows you to seamlessly export your WooCommerce orders, products, and customer data to Turpial App, enabling efficient order management and processing.

Key Features:
- Automatic order synchronization with Turpial App
- Product catalog integration
- Customer data export

== Installation ==
Upload the plugin files to the /wp-content/plugins/turpialapp-for-woo directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
- Go to WooCommerce > Settings > Integrations
- Select "Turpial App Integration"
- Enter your Turpial App API credentials
- Configure your preferred synchronization settings
- Save changes

== Usage ==
Once configured, the plugin will automatically sync your WooCommerce orders with Turpial App. You can customize which orders are sent and when synchronization occurs through the plugin settings.
Manual Order Export:
Go to WooCommerce > Orders
Select the order you wish to export
Click on the "Export to Turpial App" button

== Frequently Asked Questions ==
= How do I get API credentials for Turpial App? =
You need to register for an account at Turpial App and request API access credentials from your account dashboard.
= Is there a limit to how many orders can be synchronized? =
The synchronization capacity depends on your Turpial App subscription plan. Please refer to Turpial App's documentation for specific limitations.
= Can I sync historical orders? =
Yes, you can manually export historical orders through the WooCommerce order management interface or perform a bulk export from the plugin settings.
